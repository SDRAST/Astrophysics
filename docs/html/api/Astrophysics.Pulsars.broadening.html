broadening¶

Astrophysics.Pulsars.broadening(f, dm, model='B', dalpha=0)¶

Pulse broadeing due to interstellar scattering

The model is taken from Bhat et al., Ap.J. 605, 759 (2004), eqn. 7. The coefﬁcients, a = -6.46, b = 0.154, and c = 1.07, are only slightly different from those of Cordes & Lazio (2003), a = -6.59, b = 0.129, and c = 1.02. The frequency exponent alpha was determined by Bhat et al. to be 3.86 +/- 0.16.

:param f : frequency (GHz) :type f : float

:param dm : dispersion measure (cm^3 / pc) :type dm : float

Returns

the broadening timescale in millisec
